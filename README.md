# 🌐 Multi-Site Infrastructure – Active Directory, VPN IPsec, High Availability

## 🎯 Overview
Implementação de infraestrutura empresarial distribuída com dois sites interligados via VPN.

---

## 🏗️ Arquitetura
- Site LX: 192.168.2.0/24
- Site POR: 192.168.3.0/24
- Ligação via VPN Site-to-Site (IPsec)

---

## 🛠️ Tecnologias
- pfSense (Firewall / VPN)
- Windows Server
- Active Directory
- DHCP Failover
- DFS
- FSRM (Quotas)
- GPO
- Veeam Backup

---

## 🔎 Atividades Realizadas
- Configuração de VPN Site-to-Site (IPsec)
- Implementação de Active Directory multi-site
- Configuração de replicação entre Domain Controllers
- DHCP em alta disponibilidade (failover)
- Configuração de DFS Namespace
- Aplicação de GPOs para gestão centralizada
- Implementação de quotas com FSRM

---

## 🚨 Resultados
- Comunicação segura entre sites
- Replicação de AD validada (sem falhas)
- Alta disponibilidade de serviços
- Gestão centralizada de utilizadores e recursos
